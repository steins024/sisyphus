## How to Use This File

When starting work on a task:
1. Move the task entry from `backlog.md` to this file
2. Update the **Status** to `In Progress`
3. Update the **Updated** date
4. Add any notes about approach or blockers

When completing a task:
1. Move the task entry from this file to `completed.md`
2. Update the **Status** to `Completed`
3. Update the **Updated** date
4. Add completion notes if relevant
